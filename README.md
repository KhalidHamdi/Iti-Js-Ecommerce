# JavaScript E-commerce

## Project Description

JavaScript-based e-commerce website. offers a full suite of functionalities for browsing categories, viewing products, managing a shopping cart, and handling user authentication. The website is implemented using HTML, CSS, and JavaScript, with a focus on dynamic content loading and user interactions.


## Features

- **Dynamic Navigation and Footer**: The navigation and footer templates are dynamically loaded into the HTML pages.
- **Category and Product Display**: Categories and products are fetched from APIs and displayed dynamically.
- **User Authentication**: Basic user authentication system with login and logout functionalities.
- **Shopping Cart**: Users can add products to their cart, and the cart persists across sessions.

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/KhalidHamdi/Iti-Js-Ecommerce.git
    ```
2. Navigate to the project directory:
    ```
    cd Iti-Js-Ecommerce
    ```

## File Descriptions

- **css/style.css**: Main stylesheet for the project.
- **css/user.css**: Additional stylesheet for user-related pages.
- **js/cart.js**: Handles the shopping cart functionality.
- **js/categories.js**: Manages the fetching and display of categories.
- **js/products.js**: Manages the fetching and display of products.
- **js/script.js**: Main JavaScript file for handling general functionalities.
- **js/user.js**: Manages user authentication and related functionalities.
- **templates/footer.html**: HTML template for the footer.
- **templates/navigation.html**: HTML template for the navigation bar.
- **account.html**: User account page.
- **cart.html**: Shopping cart page.
- **category.html**: Category page for displaying products in a specific category.
- **index.html**: Main landing page of the e-commerce site.
- **login.html**: User login page.
- **logout.html**: User logout page.
- **product.html**: Product detail page.

## Usage

1. Open `index.html` in your preferred web browser.
2. Navigate through the categories and products.
3. Add products to your cart and proceed to checkout.
